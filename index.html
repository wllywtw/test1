The Open-source PlayStation 3 Emulator

RPCS3 is a multi-platform open-source PlayStation 3 emulator and debugger written in C++ for Windows, Linux, macOS and FreeBSD. Experience PS3 games like never before with enhanced performance and modern features.

Powerful Features

Lightning Fast

Optimized performance engine delivers smooth gameplay for PlayStation 3 titles with advanced CPU and GPU emulation techniques.

PS3 Compatibility

Extensive PlayStation 3 game library support with over 3000+ tested games and continuous compatibility improvements through active development.

Enhanced Graphics

Advanced rendering techniques, resolution scaling, and custom shaders enhance PS3 games beyond their original console capabilities.

Controller Support

Full DualShock 3 and DualShock 4 controller support with custom button mapping and motion control emulation for authentic PS3 gaming.

Save States

Save and load your progress anywhere in PS3 games with unlimited save slots, quick saves, and instant state switching capabilities.

Open Source

Completely free and open-source project with active community development, regular updates, and transparent development process on GitHub.
